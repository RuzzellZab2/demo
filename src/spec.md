Одна HTML-страница + один JS-файл. Без сборщиков и бэкенда.

Что должно быть:
- Поле 4x4 (16 карточек, 8 пар) и 6x6 (36 карточек, 18 пар) - выбор размера
- Перемешивание карточек при старте игры
- Открытие карточки по клику
- Логика пары: две открытые карточки сравниваются, затем либо фиксируются как пары, либо закрываются обратно
- Счетчик ходов (ход = попытка открыть пару)
- Таймер, показывающий время от начала игры до победы
- Сообщение о победе, когда все пары найдены, с отображением времени и количества ходов
- Кнопка "Новая игра" (перемешать и начать заново)
- В заголовке игры и вкладке браузера должно отображаться название Memory
- Лучший результат (минимальное время и ходы) для каждого размера поля, сохраняемый в localStorage
- Цвет карточек красный

Правила:
- Нельзя открыть третью карточку, пока не обработана пара из двух
- Найденные пары больше не кликаются
