# Memory Game

Классическая игра на память с современным дизайном и полным соответствием спецификации.

## Быстрый старт

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/yourusername/memory-game.git
   cd memory-game
   ```

2. **Запустите игру**
   ```bash
   python3 -m http.server 8080
   ```
   Откройте http://localhost:8080 в браузере.

## Возможности

### Основные функции
- Два размера поля: 4×4 (8 пар) и 6×6 (18 пар)
- Автоматическое перемешивание карточек при старте
- Таймер и счетчик ходов
- Сохранение лучших результатов в localStorage
- Адаптивный дизайн для всех устройств

### Игровой процесс
1. Выберите размер поля
2. Кликайте по карточкам, чтобы открыть их
3. Найдите все пары за минимальное время и количество ходов
4. Сравните свой результат с лучшим

### Технические особенности
- Чистый JavaScript без зависимостей
- CSS Grid для отзывчивой верстки
- CSS-анимации для плавных переходов
- Font Awesome для иконок

## Разработка

### Архитектура
Игра построена по модульному принципу:
- `game.js` содержит всю бизнес-логику
- `style.css` отвечает за внешний вид
- `index.html` обеспечивает структуру

### Ключевые функции
- `createCards()` — создает пары карточек
- `shuffleCards()` — перемешивает карточки
- `handleCardClick()` — обрабатывает клики
- `checkMatch()` — проверяет совпадения
- `saveBestScore()` — сохраняет результаты

### Тестирование
Игра полностью протестирована:
- Соответствие спецификации — 10/10 требований
- Playwright-тесты — все функции работают
- Кросс-браузерная совместимость
